## 1. Descripción del Proyecto

Este proyecto tiene como objetivo diseñar e implementar una aplicación en Python que cubra el flujo completo de trabajo de un análisis de ciencia de datos, empleando la Programación Orientada a Objetos (POO) como paradigma central de desarrollo[cite: 7]. El sistema modela las principales etapas de un pipeline de ciencia de datos: desde la carga y gestión de datos hasta la evaluación de modelos predictivos y la visualización de resultados[cite: 8]. 

El diseño prioriza la extensibilidad, la separación de responsabilidades y la reutilización de componentes mediante el uso de clases abstractas, herencia y composición[cite: 9]. [cite_start]La arquitectura está organizada en módulos independientes que se comunican a través de interfaces bien definidas, permitiendo sustituir o extender cualquier componente sin afectar al resto del sistema[cite: 10]. [cite_start]Este enfoque modular facilita el mantenimiento y la escalabilidad del proyecto a lo largo de sus diferentes fases de desarrollo

### Objetivos Específicos

* Aplicar los principios de la POO (abstracción, encapsulamiento, herencia y polimorfismo) en cada módulo del sistema.
* Diseñar una jerarquía de clases clara y extensible para los modelos de aprendizaje automático.
* Establecer una arquitectura modular que permita el desarrollo incremental del sistema.
* Documentar el diseño mediante diagramas UML antes de proceder a la implementación.
